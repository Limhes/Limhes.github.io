pyecsim.Redox¶

class pyecsim.Redox(ox: pyecsim.Species, red: pyecsim.Species, n_e: int, E: float, k_e: float, alpha: float) → pyecsim.Redox¶

Create Redox object for reaction Ox + e <-> Red with Butler-Volmer kinetics

Parameters

  • ox (pyecsim.Species) – the oxidized form of the species

  • red (pyecsim.Species) – the reduced form of the species

  • n_e (int) – number of electrons in the electron transfer step [-] (n_e >= 1)

  • E (float) – standard potential of the electron transfer step [V]

  • k_e (float) – heterogeneous rate of the electron transfer step [m/s]

  • alpha (float) – symmetry parameter of the electron transfer step [-] (0.0 <= alpha <= 1.0)

Returns

newly generated electron transfer step

Return type

pyecsim.Redox

Methods

enable()¶

Enables the electron transfer step and returns the modified Redox object

disable()¶

Disables the electron transfer step and returns the modified Redox object
